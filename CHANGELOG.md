# Changelog

All notable changes to Jake Schultz Astrophotography website/admin builds should be documented in this file.

## v2.0.0 - 2026-03-05
- Introduced Command Center as the default admin homepage.
- Restored admin route structure:
  - `/admin`
  - `/admin/editor`
  - `/admin/post`
  - `/admin/media`
- Removed the Saturn/editor password gate.
- Established semantic versioning for future releases.

## Versioning Rules
- **Major**: platform-level shift, redesign, major new system
- **Minor**: new feature or workflow added
- **Patch**: fixes, polish, cleanup, routing corrections, UI refinements
